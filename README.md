## ⚙️ Installation

Clone this repository:

```bash
git clone https://github.com/ZYU-AJA/Banking-System-OOP.git
```

Move to the project directory:

```bash
cd Banking-System-OOP
```

## ▶️ How to Run

Run the application using Python:

```bash
python Atm.py
```

or

```bash
python3 Atm.py
```
